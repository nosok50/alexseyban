My Projects

Eco City Inc.

Mobile economic strategy based on Unity 6, gameplay built around creating complex production chains. The technical foundation relies on procedural generation via Perlin noise and logistics using weighted A* pathfinding algorithms. The project architecture supports a full commercial stack, including cloud saves and custom local data encryption to protect against tampering. This is a fully independent development, spanning the cycle from writing shaders to release and support.

Solo Project Unity 6 Procedural Generation A* Pathfinding IAP Ads Cloud Saves

Wordly

Logic puzzle with an extensive dictionary of 60,000 positions. The key feature lies in a non-standard offline encryption system. The implemented algorithm generates unique keys, allowing players to exchange guessed words between devices without server or internet involvement. The project serves as a technical demonstration of complex UI architecture, DOTween animations, and algorithmic logic outside standard game loops.

Mobile Cryptography DOTween Modular Architecture

Rocky Quest

Hybrid casual project combining resource mining mechanics and Tower Defense. Development was conducted in a team format, requiring synchronization of economic upgrade cycles with active combat phases. The technical stack included a modular base building system, AI enemy navigation logic, and IAP monetization integration. The application passed closed testing stages for performance debugging and core mechanic balancing.

3D Hybrid Casual AI Pathfinding Modular Building System IAP Ads

Drilling Operations App (NDA)

Application for monitoring industrial drilling rigs. The program displays a 3D copy of the drill, synchronized in real-time with IoT sensors on the equipment. If a part overheats or requires replacement, the system immediately highlights the problem zone and issues a warning. An engineer can virtually disassemble the device on screen to study the condition of each node in detail and prevent breakdown before it occurs.

3D NDA IoT Integration Real-time Telemetry
